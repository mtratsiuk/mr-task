## Local Development

### Prerequisites

- [Node & npm](https://nodejs.org/en/download/)

### Install dependencies

```sh
npx lerna bootstrap --hoist
```

### Run tests

```sh
npm test [-- --watch]
```

### Launch `@mr/emails-input` development sandbox

```sh
cd ./packages/mr-emails-input
npm start
```
